# Stock Watcher Skill v2.0

A 股自选股管理与行情监控工具 — 多数据源增强版

## ✨ v2.0 更新亮点

| 维度 | v1.0 | v2.0 |
|------|------|------|
| 数据源 | 新浪 + 腾讯（双源） | **东方财富 + 腾讯 + 新浪（三源）** |
| 主数据源延迟 | ~107ms (新浪) | **1-3 秒 (东方财富)** |
| 技术分析 | ❌ | ✅ **20+ 指标，综合信号判断** |
| 收盘报告 | ❌ | ✅ **涨跌统计 + 持仓盈亏** |
| 风险提示 | ❌ | ✅ **涨跌停/高波动/估值预警** |
| 操盘建议 | ❌ | ✅ **基于简单规则** |
| 监控能力 | 涨跌幅 + 目标价 | **+ 涨跌停 + 成交量异常 + 预警日志** |
| 代码架构 | 各脚本独立、代码重复 | **模块化：data_source / config / watchlist_manager** |
| 依赖 | requests + bs4 + lxml | **零第三方依赖（纯标准库）** |
| 股票名称获取 | 同花顺（需 bs4） | **东方财富 API + 同花顺备选** |

## 功能概览

### 基础功能
- ✅ 自选股管理：添加、删除、查看、清空
- ✅ 行情摘要：实时价格、涨跌幅、市值
- ✅ 详细查询：买卖五档、涨跌停、估值指标
- ✅ 三数据源自动切换

### 技术分析 (新)
- 📊 MA/EMA/RSI/MACD/KDJ/BOLL/ATR 等指标
- 📝 综合信号判断（偏多/偏空/超买/超卖）

### 监控与预警 (增强)
- 🔔 涨跌幅/目标价/止损价/涨跌停预警
- 📋 预警日志记录

### 报告与建议 (新)
- 📰 每日收盘报告
- 💼 持仓盈亏
- ⚠️ 风险提示
- 💡 操盘建议

## 安装

```bash
# 方式一：SkillHub
skillhub install stock-watcher

# 方式二：手动
cp -r stock-watcher ~/.qclaw/skills/
```

## 使用示例

```bash
# 自选股管理
python scripts/add_stock.py 600519
python scripts/summarize_performance.py

# 技术分析
python scripts/technical_analysis.py 600519

# 收盘报告
python scripts/daily_report.py

# 风险提示
python scripts/risk_advice.py risk 600519
```

## 三数据源架构

东方财富（1-3s）→ 腾讯财经（5-15s）→ 新浪财经（5-20s）

自动故障转移，不混合数据源。

## 依赖

- Python 3.6+（仅标准库，零额外依赖）
- 无需 API Key
