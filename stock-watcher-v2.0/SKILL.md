### ⚠️ 用法调整（v1.0用户必读）

1. **持仓配置文件变了**: v1.0用 `portfolio.json`，v2.0同样位置但字段更丰富（新增type/差异化阈值）
2. **ETF前缀修正了**: v1.0把518开头的ETF标为sz(深圳)，v2.0修正为sh(上海)
3. **命令扩展了**: 新增 `--analyze`(三维分析) 和 `--serenity`(瓶颈筛选)
4. **输出格式变了**: 不再模糊说"建议关注"，直接给**操作+价格+理由**
5. **止损纪律**: v2.0强制-8%止损，每条买卖建议都带止损价

## 数据源策略

基于实际网络测试（西安家用网络）：
- **新浪财经** (主行情): A股/指数/国际金银，延迟1-3秒 ✅
- **东方财富 fundgz** (ETF行情): ETF实时估值+精确涨跌幅 ✅
- **东方财富 emweb** (基本面): 财报ROE/毛利率/PE/负债率 ✅
- **腾讯财经** (备行情): 买卖五档+更丰富数据 ✅
- **同花顺** (K线): 历史日K，技术分析用 ✅
- **东方财富 push2**: 当前网络不通 ❌ → 用替代方案

> perrykono-debug 的 push2 在数据中心可用，家用网络不行。我们用 fundgz+emweb 完美替代。

## 核心功能

### 1. 自选股管理
```bash
python scripts/manage.py add 600519    # 添加
python scripts/manage.py remove 600519 # 删除
python scripts/manage.py list          # 查看行情
python scripts/manage.py clear         # 清空
```

### 2. 实时行情摘要
红涨绿跌（中国习惯），支持A股/ETF/国际金银

### 3. 股票详情查询
```bash
python scripts/query_detail.py 600519  # 行情+技术分析+操盘建议
```

### 4. 7大预警规则 + 分级预警
```bash
python scripts/monitor.py              # 执行一次监控扫描
```

| 规则 | 触发条件 | 权重 | v1.0有? |
|------|----------|------|---------|
| 成本百分比 | 盈+15% / 亏-12% | ⭐⭐⭐ | ✅ |
| 日内涨跌幅 | 个股±4% / ETF±2% / 黄金±2.5% | ⭐⭐ | ✅(无差异化) |
| 成交量异动 | 放量>2倍 / 缩量<0.5倍 | ⭐⭐ | ❌ 新增 |
| 均线金叉死叉 | MA5穿MA10 | ⭐⭐⭐ | ✅ |
| RSI超买超卖 | >70 / <30 | ⭐⭐ | ❌ 新增 |
| 跳空缺口 | >1% | ⭐⭐ | ❌ 新增 |
| 动态止盈 | 盈10%+后回撤5%/10% | ⭐⭐⭐ | ❌ 新增 |

分级: 🚨紧急(多条件共振) / ⚠️警告(2条件) / 📢提醒(单条件)

### 5. 技术分析 (v2.0新增)
- MA均线系统 (5/10/20/60) → 金叉/死叉/多头/空头排列
- RSI (14日) → 超买>70/超卖<30
- MACD → 金叉/死叉/0轴上下
- 布林带 (20日, 2σ) → 突破/回归
- 成交量分析 → 量价配合/量比
- **综合评分** (-60~60) + **操盘建议** (明确文字)

### 6. 🆕 段永平选股框架 (v2.0新增)
```bash
python scripts/stock_analyzer.py --analyze 600519  # 三维综合分析
```

核心三问:
1. **赚钱吗?** → ROE>15%? 净利润增>10%? 毛利率>40%?
2. **生意模式好吗?** → 护城河(毛利率>60%=强)? 负债率<30%? 现金流健康?
3. **价格合理吗?** → PE<15低估 / 15-25合理 / 25-40偏高 / >40昂贵

评分15分制 → **明确结论**:
- 10+ → ✅ 买入 (好公司+好价格)
- 7-9 → ⚠️ 观望 (好公司+贵价格)
- 4-6 → ⚠️ 谨慎
- 0-3 → ❌ 回避

**输出格式** (学习自perrykono，禁止模糊):
```
🎯 操作: 买入
   买入价: ≤1800元
   止损价: 1656元 (-8%)
   目标价: 2160元
   理由: ROE 25%, PE 18倍(合理), MACD金叉
```

### 7. 🆕 Serenity供应链瓶颈筛选 (v2.0新增)
```bash
python scripts/stock_analyzer.py --serenity  # 六大瓶颈筛选
```

核心哲学: **不买AI龙头，狙击产业链中最短的板**

六大瓶颈环节:
- ⭐×5 先进封装 → 长电科技/通富微电
- ⭐×5 光通信/CPO → 中际旭创/天孚通信
- ⭐×5 半导体设备 → 中微公司/拓荆科技/华海清科
- ⭐×5 半导体材料 → 华大九天/南大光电
- ⭐×4 液冷散热 → 英维克/江南新材
- ⭐×4 HBM/存储 → 深科技/澜起科技

市场认知度分级: ✅已确认(等回调) / ⚠️认知差(回调即买) / 🔥最佳买点(立即关注)

### 8. 🆕 三维综合分析 (v2.0新增)
基本面(段永平) + 技术面(MA/MACD/RSI/BOLL) → 综合买卖建议

### 9. 收盘总结 (v2.0新增)
每日收盘自动生成持仓盈亏+板块表现+次日建议

### 10. 国际金银 (v2.0新增)
XAUUSD(美元/盎司) + 伦敦金hf_XAU(人民币/克) + XAGUSD

## 注意事项

- 行情1-3秒延迟，非实时tick
- 技术指标有滞后性，用于确认趋势而非预测
- 基本面数据来自最新财报（可能滞后1季度）
- **严格执行止损**，跌破止损价立即卖出
- 单只股票不超过总仓位20%
- 以上分析仅供参考，不构成投资建议

## 对话风格 (学习自perrykono)

- **犀利直接**: 结论先行，理由后置
- **禁止模糊**: 不用"建议关注"、"可以适量参与"、"或许"
- **必须明确**: 操作(买/卖/观望) + 价格(买入价/止损价/目标价) + 理由(3-5条)

❌ "茅台目前估值较低，建议关注，可以适量配置"
✅ "茅台(600519): 观望，PE 43倍(昂贵)。等回调至PE 25倍以下再买。"
